RawData 提取規則



1\. "QIP尺寸檢驗-2025.xlsx" 的 "QIP-SET UP-2025" 工作表資料需要進行去重處理，針對其中"檔案名稱"欄位的重複名稱項目進行去重，但 "QIP-SET UP-2025" 內各欄位並沒有正確的標題名稱，這是提取時的一個缺失，其他工作表有，但名稱有誤，例如 "檔名稱	 個工作表名稱	表單編碼	表單名稱"，名稱應該是"檔案名稱 	工作表名稱	表單編碼	表單名稱"



2\. "出貨檢驗-2025.xlsx" 中的工作表內容，需要去除"表單編碼"欄位為"無"的項目，以及去除"工作表名稱"欄位中包含"客戶別"、"DATE"、"出貨"等內容的項目。



3\. "押出檢驗-2025.xlsx" 去重，分成 "Setup" 與 "巡檢" 兩類，首先去除 "檔案名稱" 欄位中包含 "空白" 的項目，然後將 "工作表名稱" 欄位中含有 "SETUP" 的項目歸類為 "Setup" ，不含 "SETUP" 的項目歸類為 "巡檢"，含有 "工作表1" 的項目則不列入統計。



4\. "射出檢驗-2025\_QIP-2025(1\~10).xlsx" 去重，分成 "Setup" 與 "巡檢" 兩類，首先將 "工作表名稱" 欄位中含有 "SETUP" 的項目歸類為 "Setup" ，不含 "SETUP" 的項目歸類為 "巡檢"，其次將 "工作表名稱" 中明顯重複(如 SETUP 和 SETUP(2)， 或 250216 和 250216-1、250216-2 這種)的項目去重，這類項目只需計算一次即可。



5\. "進料檢驗-2025.xlsx" 去重， "工作表名稱" 欄位中品號相同者不必去重，例如 "RM157 56315"、"RM157 56315 (2)

"、"RM157 56315 (3)"三者視為不同項目，而含有"Sheet1"或"DATE"的則不納入計算。對於"(3)"這類不像品號的也不計算。



6\. "進料檢驗-2025" 中，先按照其中的資料夾粗分為 "原料" 與 "物料" 兩類，然後再按照物料資料夾內的名稱(如附圖)對"物料"進行細分，其中檢號為空白或是非 date code 的工作表不要列入計算。



7\. "裝配檢驗-2025.xlsx" 先不去重，其工作表按資料夾分類，對應分類將各檔案的工作表列出，不同的工作表可能對應不同的表單編號，需要人工將表單編號實施進一步對應。但須去除"表單編碼"欄位中無表單編碼的項目



8\. 最後是多個 "零組件入庫-2025\_\*.xlsx" 的檔案，其中 "工作表名稱" 欄位中應去除 "DATE" 、 "空白" 、 "範例\*"，並將 "工作表名稱" 中明顯重複 (如 SETUP 和 SETUP(2)， 或 250216 和 250216-1、250216-2 這種) 的項目去重，這類項目只需計算一次即可。所有這些提取的零組件入庫資料都應該按月份統計，其中射出A、射出C、射出D是以資料夾名稱的尾碼(後綴碼)表示月份(-A、-B、-C、....、L)，射出C(組件)、射出D(組件)與裝配A、裝配B、裝配C則是以檔名的尾碼表示月份(-A、-B、-C、....、L)。
