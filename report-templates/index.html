使用建議

  • 所有報表為 獨立 HTML 單檔，無需建置工具，直接於瀏覽器開啟即可
  • 模板 01 (Stripe) 和 05 (McKinsey) 使用 Chart.js CDN，需連網才能顯示圖表；其餘零外部依賴
  • 將對應的 data 置入 <script> 區塊即可填入其他年度資料
  • 建議使用 Chrome / Edge 列印為 PDF，即可產出正式報告
