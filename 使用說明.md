# 檔名與工作表名稱提取工具使用說明

## 1. 工具簡介
本工具專為快速提取指定資料夾內所有 Excel 檔案之「檔案名稱」與「工作表名稱」而設計，並具備**智能識別表單編碼**功能。執行後會自動建立一個新的工作表，並將提取的資料條列式呈現，方便使用者進行後續整理、分析或索引建立。

## 2. 功能特色
- **自動遍歷**：一鍵選擇資料夾，自動讀取該資料夾下所有 Excel 檔案（.xls, .xlsx, .xlsm）。
- **智能表單識別**：
    - 自動掃描工作表內容，搜尋符合 `QCxxxxx-Rxx` 格式的標準表單編碼。
    - **內建對照表**：程式內建表單名稱資料庫，自動將編碼轉換為對應的中文表單名稱（不需外部參照檔）。
- **完整提取**：列出每個檔案內的所有工作表名稱。
- **自動歸檔**：根據您選擇的資料夾名稱，自動建立並命名新的工作表來存放結果。
- **智能防呆**：
    - 自動處理 Excel 工作表命名限制（過長或特殊符號）。
    - 遇到無法開啟的檔案（如密碼保護或損壞）會自動記錄警示，不會中斷程式。

## 3. 使用步驟
1. 開啟 `檔名與工作表提取.xlsm` 檔案。
2. 若 Excel 跳出安全性警告（巨集安全性），請點選 **[啟用內容]** 以允許程式執行。
3. 按下執行按鈕（若有設置）或使用快捷鍵 `Alt + F8` 開啟巨集清單。
4. 選擇 `ExtractFileAndSheetNames_FolderAsSheet` 並點選 **[執行]**。
5. 在跳出的視窗中，選擇您要提取資料的目標資料夾，然後點選 **[確定]**。
6. 程式將自動執行，完成後會彈出成功視窗，並自動切換至新產生且已填入資料的工作表。

## 4. 輸出欄位說明
新產生的工作表將包含以下四欄資訊：
- **A欄 (檔案名稱)**：顯示 Excel 檔案的完整名稱（包含副檔名）。
- **B欄 (工作表名稱)**：顯示該檔案內包含的每一個工作表名稱。
- **C欄 (表單編碼)**：若在該工作表中偵測到如 `QC10001-R01` 的編碼，將自動填入此欄。
- **D欄 (表單名稱)**：根據 C 欄的編碼，自動填入對應的表單中文名稱（如「樣品卡」）。

## 5. 注意事項
- 本工具適用於 `.xls`, `.xlsx`, `.xlsm` 等常見 Excel 格式。
- 為確保執行速度，執行過程中畫面可能會暫時停止更新，此為正常現象。
- 若需新增或修改表單編碼對照，需進入 VBA 編輯器修改 `dictMap.Add` 相關程式碼。
